## 1. スマートキャリア診断

### 概要
5分で完了する、MBTI風の性格診断とビッグファイブ理論を組み合わせた独自のキャリア適性診断。

### 診断フロー

#### Step 1: 診断開始画面
```
画面要素:
- キャッチコピー「5分でわかる、あなたの天職と落とし穴」
- 診断開始ボタン（大きく目立つデザイン）
- 診断の信頼性アピール（500人の実データ使用）
- プライバシーポリシーへのリンク
```

#### Step 2: 基本情報入力（オプション）
```
入力項目:
- 年代（10代/20代前半/20代後半/30代/40代以上）
- 現在の状況（学生/社会人/転職活動中/その他）
- 興味のある業界（複数選択可）
※スキップ可能
```

#### Step 3: 診断質問（50問）
```
質問形式:
- 2択〜4択の選択式
- 直感的に答えられる日常的なシチュエーション
- 10問ごとにプログレスバー更新

質問カテゴリ:
1. 対人関係（10問）- E/I判定
2. 情報処理（10問）- S/N判定
3. 意思決定（10問）- T/F判定
4. ライフスタイル（10問）- J/P判定
5. ストレス耐性（10問）- 社畜化リスク判定

質問例:
Q1. 金曜日の夜、あなたは？
A) 友達と飲みに行く
B) 家でゆっくり過ごす

Q25. プロジェクトで問題が発生。あなたは？
A) データを分析して原因を特定
B) チームメンバーの意見を聞く
```

#### Step 4: 診断中画面
```
表示要素:
- ローディングアニメーション
- 「あなたの結果を分析中...」
- 診断の信頼性を示すメッセージ
- 3秒程度で結果画面へ遷移
```

### 診断結果画面

#### メインコンテンツ
```
1. MBTIタイプ表示
   - 大きなタイプ名（例：INTJ-A）
   - キャッチーな別名（例：「戦略的な建築家」）
   - タイプの特徴（3行程度の説明）

2. 社畜化リスクスコア
   - リスクメーター（0-100%）
   - 警告レベル（安全/注意/警告/危険）
   - 具体的なリスク要因TOP3

3. あなたに向いている職種TOP5
   - 職種名
   - マッチ度（%表示）
   - なぜ向いているか（1-2行）
   
4. 避けるべき環境
   - 合わない職場環境TOP3
   - 相性の悪い上司タイプ
   - ストレス要因
```

#### アクションボタン
```
- 結果をシェア（Twitter/LINE/Instagram）
- 詳細レポートを見る（登録促進）
- 哲学データベースを見る
- もう一度診断する
```

### 診断アルゴリズム

#### MBTIタイプ判定
```python
# 疑似コード
def calculate_mbti_type(answers):
    e_i_score = calculate_extraversion_score(answers[0:10])
    s_n_score = calculate_sensing_score(answers[10:20])
    t_f_score = calculate_thinking_score(answers[20:30])
    j_p_score = calculate_judging_score(answers[30:40])
    
    mbti_type = ""
    mbti_type += "E" if e_i_score > 50 else "I"
    mbti_type += "S" if s_n_score > 50 else "N"
    mbti_type += "T" if t_f_score > 50 else "F"
    mbti_type += "J" if j_p_score > 50 else "P"
    
    return mbti_type
```

#### 社畜化リスク計算
```python
def calculate_risk_score(answers, mbti_type):
    base_risk = calculate_stress_tolerance(answers[40:50])
    
    # MBTIタイプ別の補正
    type_risk_factor = {
        "ISFJ": 1.3,  # 他者への配慮が強すぎる
        "ESFJ": 1.2,  # 調和を重視しすぎる
        "INTJ": 0.8,  # 自己主張が強い
        # ... 他のタイプ
    }
    
    risk_score = base_risk * type_risk_factor.get(mbti_type, 1.0)
    return min(100, risk_score)
```

#### 職種マッチング
```python
def recommend_jobs(mbti_type, interests):
    job_database = {
        "INTJ": [
            {"job": "データサイエンティスト", "match": 95},

            # ...
        ],
        # ... 他のタイプ
    }
    
    recommendations = job_database.get(mbti_type, [])
    # 興味のある業界でフィルタリング
    if interests:
        recommendations = filter_by_interests(recommendations, interests)
    
    return recommendations[:5]
```

### データ保存
```
診断結果は以下の情報を保存:
- 診断日時
- MBTIタイプ
- 各軸のスコア（E-I, S-N, T-F, J-P）
- 社畜化リスクスコア
- 推奨職種リスト
- 回答時間（診断の信頼性指標として）
```

## 2. リアル哲学データベース

### 概要
実際に働く人々の「なぜこの仕事をしているか」という生の声を集めたデータベース。500人の面談経験を基に構築。

### コンテンツ形式

#### 1. 動画インタビュー（3分）
```
構成:
- 自己紹介（30秒）
- なぜこの仕事を選んだか（1分）
- 仕事の喜びと苦労（1分）
- 若い自分へのアドバイス（30秒）

撮影ガイドライン:
- 顔出しは任意（音声のみも可）
- 自然な語り口を推奨
- 台本なしの本音トーク
```

#### 2. 音声メッセージ（5分）
```
ポッドキャスト形式:
- より深い話ができる
- 通勤中にも聴ける
- 文字起こし付き
```

#### 3. テキストストーリー（800字）
```
構成:
- 転機となった出来事
- 価値観の変化
- 現在の仕事観
- 未来への展望
```

### データベース構造

#### タグシステム
```
必須タグ:
- MBTIタイプ（#INTJ など）
- 年代（#20代 #30代 など）
- 職種（#エンジニア #営業 など）
- 業界（#IT #金融 など）

任意タグ:
- 転職経験（#転職3回 など）
- ワークスタイル（#リモート #フレックス）
- 価値観（#成長重視 #ワークライフバランス）
```

#### 検索・フィルター機能
```
検索方法:
1. タグ検索（AND/OR検索対応）
2. フリーワード検索
3. 類似度検索（自分の診断結果に近い人）

ソート:
- 新着順
- いいね数順
- 閲覧数順
- マッチ度順
```

### UI/UXデザイン

#### リスト表示
```
カード型デザイン:
- サムネイル/アイコン
- 名前（ニックネーム可）
- MBTIタイプバッジ
- 職種・年代
- キャッチフレーズ（例：「失敗から学んだ起業家」）
- いいね数・閲覧数
```

#### 詳細表示
```
モーダルまたは専用ページ:
- コンテンツ本体（動画/音声/テキスト）
- 投稿者プロフィール
- 関連タグ一覧
- いいねボタン
- 保存ボタン
- シェアボタン
- 類似の哲学を見る
```

### 投稿システム（Phase 2以降）

#### 投稿フロー
```
1. 投稿ボタンクリック
2. 形式選択（動画/音声/テキスト）
3. ガイドラインの確認
4. コンテンツアップロード
5. タグ設定
6. プレビュー確認
7. 投稿
```

#### モデレーション
```
自動チェック:
- 不適切な言葉のフィルタリング
- 個人情報の検出
- 著作権違反の可能性

人力チェック:
- 初回投稿は必ず人力確認
- 通報機能
- コミュニティガイドライン
```

## 3. パーソナルビジョンビルダー

### 概要
診断結果と哲学データベースの内容を統合し、個人のキャリアビジョンを言語化するツール。

### 作成フロー

#### Step 1: 診断結果の振り返り
```
表示内容:
- あなたのMBTIタイプ
- 強みTOP3
- 価値観の傾向
- 推奨された職種

質問:
「診断結果を見て、納得した点は？」
「意外だった点は？」
```

#### Step 2: 哲学の選択
```
機能:
- 保存した哲学一覧表示
- 共感度の高い順にソート
- 最大5つまで選択

質問:
「特に共感した考え方は？」
「自分も大切にしたい価値観は？」
```

#### Step 3: 価値観の言語化
```
ガイド付き入力:
1. 仕事を通じて実現したいこと（100字）
2. 大切にしたい働き方（100字）
3. 避けたい状況（100字）
4. 5年後の理想の姿（100字）

入力支援:
- 例文の提示
- キーワードサジェスト
- AIによる文章補完
```

#### Step 4: ビジョンステートメント生成
```
自動生成される要素:
- タイトル（例：「技術で社会に貢献するINTJの道」）
- ビジョン文（3文程度）
- 行動指針3箇条
- 具体的なネクストステップ5つ

編集機能:
- 各要素の編集可能
- テンプレート選択
- フォント・色のカスタマイズ
```

### 出力形式

#### デジタル版
```
- Web表示用（シェア可能なURL）
- PDF版（A4縦1枚）
- 画像版（SNSシェア用）
- テキスト版（コピペ用）
```

#### ビジョンカード
```
表面:
- ビジョンステートメント
- 行動指針
- MBTIタイプアイコン

裏面:
- 参考にした哲学の要約
- 定期見直しチェックリスト
- QRコード（デジタル版へのリンク）
```

### 継続的な活用

#### リマインダー機能
```
通知タイミング:
- 作成1ヶ月後
- 3ヶ月ごと
- 転職/昇進などのライフイベント時

内容:
「ビジョンの見直し時期です」
「最近の行動は指針に沿っていますか？」
```

#### 更新履歴
```
保存される情報:
- 過去のビジョン一覧
- 変更点のハイライト
- 成長の軌跡グラフ
```

## 4. バズ機能（社畜化リスク診断）

### 概要
SNSでシェアされやすい、キャッチーで実用的な診断機能。メイン診断の一部として、または単独でも利用可能。

### 診断内容

#### リスク要因の評価
```
評価項目:
1. 残業への抵抗感の低さ
2. 断れない性格
3. 完璧主義傾向
4. 承認欲求の強さ
5. プライベートの希薄さ

スコアリング:
- 各項目0-20点
- 合計100点満点
- MBTIタイプによる補正
```

#### 結果表示

##### ビジュアル要素
```
1. リスクメーター
   - 0-30%: 安全（緑）
   - 31-60%: 注意（黄）
   - 61-80%: 警告（橙）
   - 81-100%: 危険（赤）

2. カウントダウンタイマー
   「このままだとあと○○日で限界です」
   ※計算式: (100 - リスクスコア) × 10日

3. 警告メッセージ
   危険度に応じた警告文
   例：「今すぐ有給を取るべきレベル」
```

##### 詳細分析
```
1. あなたの危険な特性TOP3
   例：
   - 「NOと言えない度: 95%」
   - 「休日も仕事を考える度: 87%」
   - 「上司の顔色を伺う度: 82%」

2. 予防アクション
   具体的な行動提案5つ
   例：
   - 「定時退社する曜日を決める」
   - 「断る練習をする」
   - 「趣味の時間を確保する」

3. 同じタイプの先輩からのアドバイス
   実体験に基づく警告とアドバイス
```

### シェア機能

#### SNS投稿テンプレート
```
Twitter:
「【社畜化リスク診断】
私のリスクスコアは○○%！
あと○○日で限界らしい...😱

特に危険なのは「○○な性格」

みんなも診断してみて！
#社畜化リスク診断 #CoolCareer
[診断URL]」

Instagram:
- 結果画像（正方形）
- ストーリー用縦長画像
- リール用動画テンプレート
```

#### バイラル要素
```
1. 友達と比較
   - ランキング表示
   - 「あなたは友達の中で○番目に危険」

2. 定期診断促進
   - 「先月より○%悪化しています」
   - 改善/悪化のグラフ

3. チャレンジ企画
   - 「#脱社畜チャレンジ」
   - 30日間の改善プログラム
```

## 5. その他のサブ機能

### 隠れ天職発見（Phase 1）
```
特徴:
- 通常の診断では出ない意外な職種提案
- 「もしも○○だったら」シミュレーション
- 成功者の転職ストーリー付き

アルゴリズム:
- 潜在的な興味・能力を分析
- 異業種転職の成功パターンマッチング
- ユーザーの想定外を意図的に提示
```

```
機能:
- 3ヶ月ごとの簡易診断（10問）
- スキルマップの更新
- 価値観の変化グラフ
- 目標達成度の記録

通知:
- 診断リマインダー
- 成長のお祝いメッセージ
- 停滞期のアドバイス
```

### AIメンタリング（Phase 3）
```
チャット機能:
- 24時間対応
- 診断結果に基づくパーソナライズ
- 具体的な悩み相談
- アクションプランの提案

学習機能:
- 過去の会話履歴から学習
- よくある質問の自動回答
- 専門カウンセラーへのエスカレーション
```

### 企業向けダッシュボード（Phase 3）
```
機能:
- チーム全体のMBTI分布
- 相性マトリックス
- 離職リスクアラート
- 採用候補者とのマッチング度

レポート:
- 月次分析レポート
- 改善提案
- ベンチマーク比較
```

## 技術仕様

### パフォーマンス要件
```
- 診断の各質問: 100ms以内でレスポンス
- 結果計算: 1秒以内
- 画像生成: 3秒以内
- 動画読み込み: プログレッシブ対応
```

### データ管理
```
- 診断データ: 暗号化して保存
- 個人情報: 最小限の収集
- 保存期間: 最終アクセスから1年
- エクスポート: GDPR対応
```

### A/Bテスト項目
```
- 質問の文言
- 結果の見せ方
- CTAボタンの配置
- シェア文言のバリエーション
```
