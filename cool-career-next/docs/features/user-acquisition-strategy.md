# Cool Career ユーザー獲得戦略

## エグゼクティブサマリー

Cool Careerは、80タイプ診断（MBTI × キャリアDNA）を就活生の「共通言語」として確立することで、強力なネットワーク効果を生み出し、持続的な成長を実現します。診断結果がTwitterプロフィールの標準装備となることで、オーガニックな拡散と高いエンゲージメントを達成します。

### 戦略の核心
- **共通言語化**: 診断結果を就活生のアイデンティティの一部に
- **ネットワーク効果**: ユーザーが増えるほど価値が向上
- **データ蓄積**: 独自データによる競争優位性の構築
- **マルチサイドプラットフォーム**: 就活生、企業、大学を繋ぐ
